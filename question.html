1 .What is Dom and What is the purpose of Dom?

Ans:Dom means Document Object Model.With this we can interact with HTML through JavaScript. Dom provide a the HTML document as a collection of node and that can be manipulated with JavaScript.
The purpose of Dom in JavaScript is it allow to dynamically access and modify the content of a web page.Dom is a fundamental things of web development

2.How to select Element through DOM?

Ans:There are 5 ways that we can select elements in a DOM using selectors

1)getElementsByTagName()
2)getElementsByClassName()
3)getElementById()
4)querySelector()
5)querySelectorAll()

3.What is Event Bubbling?

Ans:Event bubbling its like a loop .In JavaScript where an event that is triggered on a child element like a button is passed up to the parent elements like a container. This happens automatically and you can handle the event on the parent element instead of the child element if you want to. You can stop the event loop by useing stopPropagation() method
